React

A JavaScript library for building user interfaces

Learn Once, Write Anywhere

We dont make assumptions about the rest of your technology stack, so you can develop new features in React without rewriting existing code. React can render on the server using Node and power mobile apps using React Native

A Simple Component

React components implement a render() method that takes input data and returns what to display. This example uses an XML-like syntax called JSX. Input data that is passed into the component can be accessed by render() via this.props.

JSX is optional and not required to use React. Try the Babel REPL to see that raw JavaScript code produced by the JSX compilation step.
